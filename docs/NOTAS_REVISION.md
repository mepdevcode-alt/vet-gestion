# Notas de revisión — llamada Discord

## Observaciones de UX / flujo

### 1. El dueño no puede cargar sus propias mascotas
- **Problema:** Un cliente nuevo no puede registrar su mascota por sí solo. El rol Dueño solo puede *ver* sus mascotas, no crearlas. Para solicitar un turno primero necesita que un admin o recepcionista le cargue la mascota.
- **Propuesta:** Permitir que el rol Dueño pueda dar de alta sus propias mascotas.
