# PROJECT_CONTEXT.md - Contexto del Proyecto

## Propósito
Sistema de gestión integral para clínicas veterinarias que permite el seguimiento médico de mascotas y la localización en tiempo real mediante dispositivos GPS.

## Objetivos Clave
1. **Confianza del Dueño:** Proporcionar una herramienta visual (mapa) para saber dónde está su mascota.
2. **Eficiencia Médica:** Historial clínico digitalizado, accesible y seguro.
3. **Escalabilidad:** Arquitectura preparada para manejar múltiples clínicas y cientos de collares GPS simultáneos.

## Valor Diferencial
A diferencia de otros sistemas de gestión, este integra **WebSockets** y **OpenStreetMap** para una experiencia de usuario fluida y sin costos de licenciamiento de mapas (como Google Maps).

## Audiencia
- Administradores de clínicas.
- Médicos Veterinarios.
- Dueños de mascotas (clientes finales).
